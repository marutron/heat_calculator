#### Скрипт для подготовки справки по энерговыделению в отсеках БВ во время ремонта.

Для запуска:

1. Разместить в директории `input` файлы:
    1) `initial_state` - файл состояния ТОПАЗ на начало ремонта
    2) `stage_3.mp` - файл МП выгрузка ТВС из АЗ в БВ
    3) `stage_5.mp` - файл МП загрузка ТВС из БВ в АЗ
    4) `otvs.mp` - файл МП для вывоза ОТВС

2. Заполнить управляющий файл - `controller.py` в директории `input` своими данными, например:

```python
block_number = 2  # номер блока
begin_date = "06.11.2025"  # дата начала справки
end_date = "20.12.2025"  # последняя дата справки

# выгрузка ТВС из АЗ в БВ
stage_3_begin = "15.11.2025 11:00"
stage_3_end = "19.11.2025 22:00"

# загрузка ТВС из БВ в АЗ
stage_5_begin = "30.11.2025 10:00"
stage_5_end = "03.12.2025 12:00"

# отправка ОТВС
otvs_begin = "03.12.2025 12:00"
otvs_end = "05.12.2025 13:00"
```

Скрипт автоматически рассчитает время выполнения операций и заполнит итоговую таблицу `output/table.odt`

Запустить скрипт из командной строки:
```commandline
python3 main.py
```
